# Flow Image Stepper

Google Labs Flow에서 캐릭터 시트와 장면 이미지를 순서대로 생성하고, 저장된 결과를 다시 찾아 다운로드하는 Chrome 확장 프로그램입니다.

## 주요 기능

- 긴 프롬프트 묶음에서 캐릭터와 장면 목록을 자동 분리
- 캐릭터 시트를 먼저 생성하고 프로젝트별 참조 보관함에 저장
- 저장된 캐릭터 참조를 사용해 장면 이미지를 순서대로 생성
- 프로젝트별로 캐릭터 참조, 장면 진행 상태, 저장된 장면 결과를 분리
- 같은 프로젝트에서 2차/3차 장면을 추가할 때 기존 마지막 장면 번호 다음부터 이어서 저장
- 개별 캐릭터 또는 장면만 재생성 큐에 추가
- 저장된 장면 이미지를 순서대로 일괄 다운로드

## 설치 방법

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위의 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
4. 이 폴더를 선택합니다.
5. 파일을 수정한 뒤에는 `chrome://extensions`에서 확장 프로그램을 새로고침합니다.

## 사용 흐름

1. Google Labs Flow 프로젝트 화면을 엽니다.
2. 확장 프로그램 사이드패널을 엽니다.
3. 전체 프롬프트 원문을 붙여넣거나 빌더에서 캐릭터/장면을 작성합니다.
4. 모델, 장면 생성 수, 화면 비율을 선택합니다.
5. `프롬프트 목록 만들기`로 캐릭터와 장면 목록을 준비합니다.
6. `다음 캐릭터 시트 생성`으로 필요한 캐릭터 참조를 저장합니다.
7. `다음 장면 이미지 생성`으로 장면 이미지를 생성합니다.
8. 필요한 경우 개별 프롬프트 수정, 재생성 큐, 한 장면 뒤로 기능을 사용합니다.
9. `전체 장면 이미지 다운로드`로 저장된 결과를 내려받습니다.

## 프로젝트별 상태

확장 프로그램은 Flow URL의 `/project/{id}` 값을 기준으로 상태를 분리합니다.

- 새 프로젝트에서는 이전 프로젝트의 캐릭터 참조와 장면 번호를 자동으로 가져오지 않습니다.
- 같은 프로젝트에서 새 장면 묶음을 추가하면 기존 마지막 장면 번호 다음부터 이어집니다.
- 프로젝트 ID가 아직 없는 새 Flow 화면에서는 이전 프로젝트 상태를 fallback으로 사용하지 않습니다.
- `입력 지우기`는 현재 작업 목록을 초기화하지만, 같은 프로젝트의 마지막 장면 번호는 이어가기용으로 보존합니다.

## 입력 예시

```text
캐릭터 시트 프롬프트 - 연화_CS-01
이름: 연화
프롬프트:
연화의 캐릭터 시트 이미지 프롬프트...

캐릭터 시트 프롬프트 - 민준_CS-02
이름: 민준
프롬프트:
민준의 캐릭터 시트 이미지 프롬프트...

장면 이미지 프롬프트 1
배치: 비 오는 골목
등장인물: 연화_CS-01
프롬프트:
비 오는 골목을 걷는 연화 장면 이미지 프롬프트...

장면 이미지 프롬프트 2
배치: 카페 창가
등장인물: 연화_CS-01, 민준_CS-02
프롬프트:
카페 창가에서 만나는 연화와 민준 장면 이미지 프롬프트...
```

기존 형식도 사용할 수 있습니다.

````text
Character ID: 연화_CS-01

```
character sheet prompt...
```

Image 1/6
등장인물: 연화_CS-01

```
scene prompt...
```
````

## 설정

- 모델: `Nano Banana Pro`, `Nano Banana 2`, `Imagen 4`
- 장면 생성 수: `x1` ~ `x4`
- 화면 비율: `16:9`, `4:3`, `1:1`, `3:4`, `9:16`

`Nano Banana Pro`는 안정성을 위해 장면 생성 수를 최대 `x2`로 제한합니다.
