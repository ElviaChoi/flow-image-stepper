# Flow Image Stepper ✨

Google Labs Flow에서 캐릭터 시트와 장면 이미지를 순서대로 생성하기 위한 Chrome 확장 프로그램입니다. 캐릭터 참조 이미지를 저장해두고 장면 생성에 다시 사용해, 여러 장면에서 캐릭터 일관성을 유지하는 데 초점을 둡니다.

## 주요 기능 🎯

- 🧾 전체 프롬프트를 캐릭터/장면 목록으로 분리
- 🧍 캐릭터 시트를 한 명씩 생성하고 참조 이미지로 저장
- 🎬 저장된 캐릭터 참조를 붙여 장면 이미지를 순서대로 생성
- ✏️ 결과가 마음에 들지 않는 캐릭터/장면의 프롬프트만 수정 후 다시 생성 준비
- 🧠 Nano Banana Pro, Nano Banana 2, Imagen 4 모델 선택
- 📌 진행 요약, 한국어 로그, 최신 결과 저장, 장면 이미지 다운로드 지원

## 사용 흐름 🚦

1. 🌊 Google Labs Flow 페이지를 엽니다.
2. 🧩 확장 프로그램 사이드패널을 엽니다.
3. 📝 **전체 프롬프트 원본**에 캐릭터 시트와 장면 프롬프트를 붙여넣습니다.
4. ⚙️ 모델, 장면 생성 수, 화면 비율을 선택합니다.
5. 🧾 **프롬프트 목록 만들기**를 눌러 캐릭터/장면 목록을 만듭니다.
6. 🧍 **다음 캐릭터 시트 생성**으로 캐릭터 참조 이미지를 차례대로 만듭니다.
7. 🎬 **다음 장면 이미지 생성**으로 장면 이미지를 차례대로 만듭니다.
8. ✏️ 필요한 경우 **개별 프롬프트 수정**에서 해당 캐릭터나 장면만 수정해 다시 생성 준비합니다.
9. 📥 생성이 끝나면 **장면 이미지 다운로드**로 저장된 장면 이미지를 내려받습니다.

## 버튼 설명 🧭

- 🧾 **프롬프트 목록 만들기**: 붙여넣은 전체 프롬프트를 캐릭터/장면 목록으로 나눕니다.
- 🧹 **작업 초기화**: 저장된 원본, 진행 상태, 참조 이미지 기록, 장면 결과 기록을 지웁니다.
- 🧍 **다음 캐릭터 시트 생성**: 다음 캐릭터 시트를 생성하고 참조 이미지로 저장합니다.
- 🎬 **다음 장면 이미지 생성**: 필요한 캐릭터 참조를 붙인 뒤 다음 장면 이미지를 생성합니다.
- 🔁 **저장된 캐릭터 참조 불러오기**: 현재 Flow 프로젝트에 저장된 캐릭터 참조를 다시 불러옵니다.
- 💾 **최신 장면 결과 저장**: Flow 화면에 이미 생성된 최신 이미지를 현재 장면 결과로 저장합니다.
- ↩️ **이전 장면 다시 생성**: 직전 장면으로 돌아가 해당 장면 결과를 다시 만들 수 있게 합니다.
- 📦 **캐릭터 참조 백업(JSON)**: 프로젝트별 캐릭터 참조 데이터를 JSON 파일로 백업합니다.
- ♻️ **캐릭터 참조 복원(JSON)**: 백업한 JSON 파일에서 캐릭터 참조 데이터를 복원합니다.
- ✏️ **수정하고 다시 생성 준비**: 선택한 캐릭터/장면 프롬프트를 저장하고, 해당 항목부터 다시 생성할 수 있게 준비합니다.
- 📥 **장면 이미지 다운로드**: 저장된 장면 이미지들을 파일명 규칙에 맞춰 다운로드합니다.

## 모델과 설정 ⚙️

- 선택 가능한 모델: **Nano Banana Pro**, **Nano Banana 2**, **Imagen 4**
- 장면 생성 수: `x1`부터 `x4`까지 선택할 수 있습니다.
- 화면 비율: `16:9`, `4:3`, `1:1`, `3:4`, `9:16`을 선택할 수 있습니다.
- **Nano Banana Pro**는 사용량 제한에 걸리기 쉬워 장면 생성 수가 최대 `x2`로 제한됩니다.

## 설치 🧩

1. Chrome에서 `chrome://extensions`를 엽니다.
2. **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램 로드**를 누릅니다.
4. 이 저장소를 내려받은 폴더를 선택합니다.

코드를 수정한 뒤에는 `chrome://extensions`에서 확장 프로그램을 다시 로드해야 합니다.
