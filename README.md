# Flow Image Stepper ✨

Google Labs Flow에서 캐릭터 시트와 장면 이미지를 순서대로 생성하기 위한 Chrome 확장 프로그램입니다. 캐릭터 참조 이미지를 저장해두고 장면 생성에 다시 사용해, 여러 장면에서 캐릭터 일관성을 유지하는 데 초점을 둡니다.

## 주요 기능 🎯

- 전체 프롬프트를 캐릭터/장면 목록으로 분리
- 캐릭터 시트를 한 명씩 생성하고 참조 이미지로 저장
- 저장된 캐릭터 참조를 붙여 장면 이미지를 순서대로 생성
- 결과가 마음에 들지 않는 캐릭터/장면의 프롬프트만 수정 후 다시 생성
- 진행 요약, 최신 결과 저장, 장면 이미지 다운로드 지원

## 사용 흐름 🚦

1. Google Labs Flow 페이지를 엽니다.
2. 확장 프로그램 사이드패널을 엽니다.
3. **전체 프롬프트 원본**에 캐릭터 시트와 장면 프롬프트를 붙여넣습니다.
4. **목록 만들기**를 눌러 캐릭터/장면 목록을 만듭니다.
5. **다음 캐릭터 생성**으로 캐릭터 참조 이미지를 차례대로 만듭니다.
6. **다음 장면 생성**으로 장면 이미지를 차례대로 만듭니다.
7. 필요한 경우 **개별 프롬프트 수정**에서 해당 캐릭터나 장면만 수정해 다시 생성합니다.
8. 생성이 끝나면 **장면 이미지 다운로드**로 저장된 장면 이미지를 내려받습니다.

## 버튼 설명 🧭

- **목록 만들기**: 붙여넣은 전체 프롬프트를 캐릭터/장면 목록으로 나눕니다.
- **전체 작업 초기화**: 저장된 원본, 진행 상태, 참조 이미지 기록, 장면 결과 기록을 지웁니다.
- **다음 캐릭터 생성**: 다음 캐릭터 시트를 생성하고 참조 이미지로 저장합니다.
- **다음 장면 생성**: 필요한 캐릭터 참조를 붙인 뒤 다음 장면 이미지를 생성합니다.
- **최신 결과 저장**: Flow 화면에 이미 생성된 최신 이미지를 현재 장면 결과로 저장합니다.
- **장면 다시 생성**: 직전 장면으로 돌아가 해당 장면 결과를 다시 만들 수 있게 합니다.
- **장면 이미지 다운로드**: 저장된 장면 이미지들을 파일명 규칙에 맞춰 다운로드합니다.

## 설치 🧩

1. Chrome에서 `chrome://extensions`를 엽니다.
2. **개발자 모드**를 켭니다.
3. **압축해제된 확장 프로그램 로드**를 누릅니다.
4. 이 폴더를 선택합니다.

```text
C:\Users\user\Desktop\Flow Image Stepper
```

코드를 수정한 뒤에는 `chrome://extensions`에서 확장 프로그램을 다시 로드해야 합니다.
