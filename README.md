# Flow Image Stepper ✨

Google Labs Flow에서 캐릭터 시트와 장면 이미지를 순서대로 생성하기 위한 Chrome 확장 프로그램입니다.

긴 프롬프트를 캐릭터 단계와 장면 단계로 나누고, 저장된 캐릭터 참조를 재사용하면서 필요한 항목만 다시 생성할 수 있게 돕습니다.

## 주요 기능 🚀

- 전체 프롬프트를 캐릭터와 장면 목록으로 분리
- 캐릭터 시트를 순서대로 생성하고 참조 이미지로 저장
- 저장된 캐릭터 참조를 사용해 장면 이미지를 순서대로 생성
- 개별 프롬프트 수정 후 특정 항목만 다시 생성하거나 해당 지점부터 다시 생성
- 저장된 장면 이미지 다운로드
- 진행 상태와 저장 상태 확인

## 사용 흐름 🧭

1. Google Labs Flow를 엽니다.
2. 확장 프로그램 팝업을 엽니다.
3. 프롬프트 원본을 붙여넣습니다.
4. 모델, 장면 생성 수, 화면 비율을 선택합니다.
5. `프롬프트 목록 만들기`를 눌러 캐릭터와 장면 목록을 준비합니다.
6. `다음 캐릭터 시트 생성`으로 캐릭터 참조를 만듭니다.
7. `다음 장면 이미지 생성`으로 장면 이미지를 생성합니다.
8. 필요하면 `개별 프롬프트 수정`이나 `고급 도구`를 사용합니다.
9. `전체 장면 이미지 다운로드`로 결과를 저장합니다.

## 입력 예시 📝

````text
Character ID: 한수_CS-01

```
character sheet prompt...
```

Image 1/6
등장인물: 한수_CS-01

```
scene prompt...
```
````

장면만 있는 프롬프트도 사용할 수 있습니다. 이 경우 `등장인물:` 줄에 캐릭터 ID를 적어두면 저장된 캐릭터 참조를 다시 연결할 수 있습니다.

## 지원 설정 ⚙️

- 모델: `Nano Banana Pro`, `Nano Banana 2`, `Imagen 4`
- 장면 생성 수: `x1` ~ `x4`
- 화면 비율: `16:9`, `4:3`, `1:1`, `3:4`, `9:16`

`Nano Banana Pro`는 팝업에서 `x2`까지로 제한됩니다.

## 설치 방법 📦

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
4. 이 폴더를 선택합니다.

파일을 수정한 뒤에는 `chrome://extensions`에서 확장 프로그램을 새로고침하면 됩니다.
