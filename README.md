# Flow Image Stepper ✨

Google Labs Flow에서 캐릭터 시트와 장면 이미지를 순서대로 생성하기 위한 Chrome 확장 프로그램입니다.

하나의 긴 프롬프트를 캐릭터 단계와 장면 단계로 나누고, 저장된 캐릭터 참조 이미지를 재사용하면서 필요한 부분만 다시 생성할 수 있게 돕습니다.

## 주요 기능 🚀

- 전체 프롬프트를 캐릭터 시트와 장면 목록으로 분리
- 캐릭터 시트를 순서대로 생성하고 참조 이미지로 저장
- 저장된 캐릭터 참조를 붙여 장면 이미지를 순서대로 생성
- 현재 Flow 프로젝트에 저장된 캐릭터 참조 다시 불러오기
- 특정 캐릭터나 장면 프롬프트만 수정하고 해당 지점부터 다시 생성
- `등장인물:` 정보가 있는 장면만 입력해도 저장된 캐릭터 참조 불러오기 지원
- `Image 1/6` 같은 장면 헤더가 없는 장면 프롬프트도 `등장인물:` 기준으로 참조 캐릭터 인식
- 캐릭터 참조 정보가 없는 일반 텍스트 입력은 간단 장면 모드로 처리
- 저장된 전체 장면 이미지 다운로드
- 진행 요약, 저장 상태, 재생성 포인트 확인
- 작업 초기화 전 확인 팝업 표시

## 사용 흐름 🧭

1. Google Labs Flow를 엽니다.
2. 확장 프로그램 팝업을 엽니다.
3. 전체 프롬프트 원본 입력칸에 내용을 붙여넣습니다.
4. 모델, 장면 생성 수, 화면 비율을 선택합니다.
5. `프롬프트 목록 만들기`로 캐릭터와 장면 목록을 준비합니다.
6. `다음 캐릭터 시트 생성`으로 캐릭터 참조를 순서대로 만듭니다.
7. `다음 장면 이미지 생성`으로 장면 이미지를 순서대로 생성합니다.
8. 필요한 경우 `개별 프롬프트 수정`에서 한 항목만 수정하고 다시 이어서 생성합니다.
9. `전체 장면 이미지 다운로드`로 저장된 장면 결과를 내려받습니다.

## 입력 방식 📝

전체 프롬프트를 한 번에 붙여넣으면 확장 프로그램이 캐릭터 단계와 장면 단계를 나누어 순서대로 진행할 수 있게 준비합니다.

- 캐릭터 시트와 장면이 함께 있는 입력을 사용할 수 있습니다.
- 장면만 있는 입력도 사용할 수 있습니다. 이때 `등장인물:` 줄에 `한수_CS-01` 같은 캐릭터 ID가 있으면 저장된 캐릭터 참조를 다시 불러올 수 있습니다.
- 코드블록 시작 줄에 `id="t9x2qp"` 같은 메타데이터가 붙어 있어도 실제 프롬프트에는 포함하지 않고 제거합니다.
- 세부 입력 규칙은 확장 프로그램 UI 기준으로 맞춰 사용하는 것을 권장합니다.

예시:

````text
Character ID: 한수_CS-01

```id="t9x2qp"
character sheet prompt...
```

Image 1/6
등장인물: 한수_CS-01

```id="d4n7sa"
scene prompt...
```
````

## 지원 설정 ⚙️

- 모델: `Nano Banana Pro`, `Nano Banana 2`, `Imagen 4`
- 장면 생성 수: `x1` ~ `x4`
- 화면 비율: `16:9`, `4:3`, `1:1`, `3:4`, `9:16`

참고로 `Nano Banana Pro`는 팝업에서 `x2`까지로 제한됩니다.

## 설치 방법 📦

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
4. 이 폴더를 선택합니다.

파일을 수정한 뒤에는 `chrome://extensions`에서 확장 프로그램을 다시 새로고침하면 됩니다.
