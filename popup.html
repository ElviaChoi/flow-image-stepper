Flow Image Stepper ✨

캐릭터 일관성을 유지할 수 있도록 캐릭터 시트와 장면 이미지를 Flow에서 단계별로 생성합니다.

준비

프롬프트를 파싱해서 캐릭터/장면 순서를 준비합니다.

생성

캐릭터 시트를 먼저 만들고, 저장된 참조로 장면을 생성합니다.

내보내기

저장된 장면 이미지를 파일로 다운로드합니다.

고급 도구

오류 복구나 데이터 이전이 필요할 때만 사용하세요.

진행 요약

아직 목록이 없습니다.

개별 프롬프트 수정

프롬프트를 수정하고 다시 생성 준비를 누르면 해당 캐릭터나 장면부터 다시 만들 수 있습니다.
